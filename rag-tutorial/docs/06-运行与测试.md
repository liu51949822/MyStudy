# 第 6 课：运行与测试

## 快速启动
```bash
cd rag-tutorial
npm install
cp .env.local.example .env.local  # 编辑填入你的配置
npm run dev
```

## 测试 API
```bash
# 存入知识
curl -X POST http://localhost:3000/api/ingest \
  -H "Content-Type: application/json" \
  -d '{"content": "RAG 是检索增强生成的缩写。"}'

# 提问
curl -X POST http://localhost:3000/api/query \
  -H "Content-Type: application/json" \
  -d '{"question": "什么是 RAG？"}'
```

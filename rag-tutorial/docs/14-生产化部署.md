## 14.2 缓存（Cache）

```typescript
// LRU 缓存 — 最近最少使用淘汰策略
const cache = new LRUCache(maxSize=100, ttlMs=5min);

// 使用前先查缓存
let result = cache.get(key);
if (!result) {
  result = await expensiveOperation();
  cache.set(key, result);
}
```

**缓存什么？**
- 嵌入向量（相同的文本生成相同的向量，100% 命中）
- 搜索结果（短时间内相同问题会重复搜索）

## 14.3 限流（Rate Limiting）

```typescript
// 每个 IP 每分钟最多 30 次请求
const result = checkRateLimit(ip, 30, 60 * 1000);
if (!result.allowed) {
  return 429 Too Many Requests;
}
```

**为什么要限流？**
- DeepSeek API 有费率限制
- 防止滥用
- 保护数据库

## 14.4 监控（Telemetry）

```typescript
const timer = startTimer("query");
// ... 执行业务逻辑 ...
timer.end({ result: "success" });

// 控制台输出：
// [query] 1234ms { result: "success" }
// 如果 > 5000ms，会标记 ⚠️
```

## 14.5 重试（Retry）

```typescript
const result = await withRetry(
  () => openai.embeddings.create({...}),
  maxRetries: 3,
  baseDelayMs: 1000
);
// 失败后自动重试：等 1s → 等 2s → 等 4s
```

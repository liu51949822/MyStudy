# 第 2 课：代码结构与核心逻辑

## 三层架构
```
UI 层 (page.tsx)          → 用户界面
API 层 (api/*/route.ts)   → 请求响应
业务逻辑层 (lib/rag.ts)   → RAG 编排
基础设施层 (lib/db.ts, lib/embeddings.ts) → DB + AI
```

## 三个核心文件
- **lib/db.ts**: PostgreSQL + pgvector 连接和操作
- **lib/embeddings.ts**: DeepSeek 嵌入向量生成和 AI 问答
- **lib/rag.ts**: RAG 工作流编排（ingest + query）

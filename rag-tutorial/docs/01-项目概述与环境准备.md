# 第 1 课：项目概述与环境准备

## 什么是 RAG？
RAG = Retrieval-Augmented Generation（检索增强生成）。
通俗说：让 AI 在回答问题前先去查资料。

## 技术栈
- Next.js 14 + TypeScript — Web 框架
- PostgreSQL + pgvector — 数据库 + 向量存储
- DeepSeek API — 嵌入向量 + AI 回答

## 环境准备
1. Node.js >= 18
2. PostgreSQL + pgvector 扩展
3. DeepSeek API Key
4. 配置 .env.local

## 项目结构
```
rag-tutorial/
├── app/
│   ├── api/ingest/route.ts    # 知识入库 API
│   ├── api/query/route.ts     # 知识问答 API
│   └── page.tsx               # 主页面
├── lib/
│   ├── db.ts                  # 数据库操作
│   ├── embeddings.ts          # DeepSeek 嵌入 + AI 问答
│   └── rag.ts                 # RAG 核心工作流
├── docs/                      # 教程文档
├── .env.local.example         # 环境变量模板
└── package.json
```

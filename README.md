# RAG 入门教程 — 全栈生产级 RAG

## 相对于 rag_7 的新增功能
- ⚡ **LRU 缓存**: 缓存嵌入向量和搜索结果，减少 API 调用
- 🚦 **请求限流**: 滑动窗口限流，防止滥用
- 📊 **遥测监控**: 记录请求耗时，慢查询标记
- 🔄 **自动重试**: 指数退避重试失败请求

## 全系列总结
| 分支 | 核心能力 | 知识点 |

| rag_1 | 基础 RAG | 入库 + 搜索 + 问答 |

| rag_5 | MCP 集成 | 工具调用 + Function Calling |
| rag_6 | Skill 工程 | 技能系统 + Prompt 模板 |

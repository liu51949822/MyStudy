# 📖 PHP 第 8 课：生产部署与安全

## Security Headers
```php
header('X-Content-Type-Options: nosniff');
header('X-Frame-Options: DENY');
```

## CSRF 保护
```php
// 生成 token
$_SESSION['csrf_token'] = bin2hex(random_bytes(32));

// 验证 token
if (!hash_equals($_SESSION['csrf_token'], $token)) {
    exit('CSRF token mismatch');
}
```

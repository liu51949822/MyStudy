# 📖 PHP 第 5 课：用户认证系统

## Session 机制
```php
session_start();
$_SESSION['user'] = $username;
```

## 密码哈希
```php
// 注册时
$hash = password_hash($password, PASSWORD_BCRYPT);

// 登录时
if (password_verify($password, $hash)) {
    // 登录成功
}
```

## Auth 中间件
```php
if (!isset($_SESSION['user'])) {
    header('Location: /login');
    exit;
}
```

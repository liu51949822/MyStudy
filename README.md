# PHP 递进式教程

从零开始的 PHP 学习路线，每个分支一个难度等级。

## 分支路线

| 分支 | 难度 | 内容 | 项目 |
|------|------|------|------|
| php_1 | ★☆☆☆☆ | 变量/数组/函数/表单/文件 | Todo List (文件存储) |
| php_2 | ★★☆☆☆ | PDO/MySQL/CRUD/分页 | Todo List (数据库) |
| php_3 | ★★★☆☆ | MVC 架构/路由/模板 | MVC Todo |
| php_4 | ★★★☆☆ | 认证/Session/权限 | 用户系统 |
| php_5 | ★★★★☆ | REST API/JSON/Token | API 服务 |
| php_6 | ★★★★☆ | DI/测试/设计模式 | 重构优化 |
| php_7 | ★★★★★ | 安全/部署/Docker | 生产上线 |

## 快速开始

```bash
# php_1: 基础版
php -S localhost:8080 -t php-tutorial/public
```

## 教学文档
每个分支的 `php-tutorial/docs/` 目录下包含完整教学文档。

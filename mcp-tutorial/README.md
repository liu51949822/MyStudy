# MCP 递进式教程

从零开始的 MCP (Model Context Protocol) 学习路线。

## 分支路线
| 分支 | 难度 | 内容 |
|------|------|------|
| mcp_1 | ★☆☆☆☆ | MCP 基础概念 + 第一个 Server |
| mcp_2 | ★★☆☆☆ | 多工具 + 参数校验 |
| mcp_3 | ★★★☆☆ | Resources + Prompts |
| mcp_4 | ★★★☆☆ | 数据传输 (stdio/SSE) |
| mcp_5 | ★★★★☆ | MCP Client 开发 |
| mcp_6 | ★★★★☆ | AI 模型集成 |
| mcp_7 | ★★★★★ | 生产级 MCP Server |

## 启动
```bash
cd mcp-tutorial
npm install
npm run dev
```

# 📖 MCP 第 2 课：多工具与参数校验

## 多个工具注册
```typescript
tools: [
  { name: "greeting", ... },
  { name: "calculator", ... },
  { name: "echo", ... },
]
```

## 参数类型与校验
MCP 使用 JSON Schema 定义参数：
- string / number / boolean
- enum 枚举值
- required 必填字段

## 工具路由
```typescript
switch (name) {
  case "greeting": ...
  case "calculator": ...
}
```

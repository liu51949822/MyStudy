# 📖 PHP 第 7 课：设计模式与依赖注入

## 依赖注入容器
```php
$container = new Container();
$container->set(Todo::class, fn() => new Todo());
$todoService = $container->get(TodoService::class);
```

## 服务层模式
业务逻辑从 Controller 分离到 Service 层：
- Controller: 只处理 HTTP 请求
- Service: 处理业务逻辑
- Model: 处理数据持久化

## 单一职责原则
每个类只负责一件事。

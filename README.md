# RAG 入门教程 — 全栈生产级 RAG

## 相对于 rag_7 的新增功能
- ⚡ **LRU 缓存**: 缓存嵌入向量和搜索结果，减少 API 调用
- 🚦 **请求限流**: 滑动窗口限流，防止滥用
- 📊 **遥测监控**: 记录请求耗时和关键指标
- 🔄 **自动重试**: 指数退避重试失败请求

## 新增文件
| 文件 | 说明 |
|------|------|
| `lib/cache.ts` | LRU 缓存（嵌入向量 + 搜索） |
| `lib/rate-limiter.ts` | 滑动窗口限流器 |
| `lib/telemetry.ts` | 请求监控和日志 |
| `lib/retry.ts` | 指数退避重试 |
| `app/api/middleware.ts` | API 限流中间件 |

## 核心指标
- **缓存命中**: 相同文本的嵌入向量直接返回
- **限流策略**: 每个 IP 每分钟最多 30 次
- **重试策略**: 3 次重试，等待时间 1s → 2s → 4s
- **慢查询标记**: 超过 5s 的请求会被标记 ⚠️
